# Drums of War

Multi-expansion World of Warcraft private server platform with a unified website, character transfer pipeline, and admin panel.

**Three servers, one account:**
- **Classic** (1.12.1) — VMaNGOS
- **The Burning Crusade** (2.4.3) — CMaNGOS
- **Wrath of the Lich King** (3.3.5a) — AzerothCore

## Features

- **Unified Website** — Next.js dark-themed site with server status, news, armory, talent trees
- **Single Account System** — One registration creates accounts on all three servers
- **Character Transfer** — Classic → TBC → WotLK progression pipeline
- **Armory** — Character profiles, equipment, talent trees with real spell values
- **Admin Panel** — Server management, logs, news publishing, Docker control
- **Email Verification** — Registration with Resend email service
- **Auction House Browser** — Live AH data across all servers
- **Honor Leaderboard** — PvP rankings with faction tabs

## Requirements

- Docker + Docker Compose
- Git
- A domain name (for HTTPS via Traefik)
- Game client data (maps/vmaps/mmaps/dbc) for each expansion
- Optional: Resend API key (for email verification)

## Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/semorgana/drums-of-war.git
cd drums-of-war

# 2. Run setup (clones upstream repos, generates .env)
./setup.sh

# 3. Edit configuration
nano .env    # Set your domain, IP, and API keys

# 4. Place game data
# Copy maps/vmaps/mmaps/dbc into data/classic/, data/tbc/, data/wotlk/

# 5. Build and start everything
./update.sh --build-all
```

## Configuration

All configuration is done through `.env` files. See `.env.example` for all available options.

| Variable | Description |
|---|---|
| `SERVER_PUBLIC_IP` | Your domain or IP for realmlist |
| `MARIADB_ROOT_PASSWORD` | Database password (auto-generated by setup.sh) |
| `MW_DOMAIN` | Website domain |
| `JWT_SECRET` | JWT signing key (auto-generated by setup.sh) |
| `RESEND_API_KEY` | Resend.com API key for email verification |

## Updating

```bash
# Pull latest code + upstream repos
./update.sh

# Pull + rebuild + restart containers
./update.sh --build-all
```

## Architecture

- **Traefik** — reverse proxy with automatic HTTPS
- **3 Game Servers** — each in its own Docker stack with separate DB
- **Unified Website** — Next.js connecting to all 3 databases
- **Auth Service** — SRP6 password verification across patches
- **Transfer Pipeline** — sequential Classic → TBC → WotLK character migration
